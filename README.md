# Анализ видео через ChatGPT Vision

Скрипт для автоматического анализа видео с использованием OpenAI Vision API.

## ⚠️ Важно
- **Требуется API ключ OpenAI** (хранится в `api_key.txt`)
- **Платный сервис** — каждый кадр = отдельный запрос
- Рекомендуется использовать **не более 1 кадра в секунду**

## Установка
```bash
pip install openai opencv-python
